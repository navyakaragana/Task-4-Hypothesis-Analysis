Student Data Analysis Project
Objective
Analyze student demographic data to identify patterns and support decision-making.

Key Insights
Balanced gender distribution
Stable academic year distribution
Consistent age range
Variation in course enrollment
Hypothesis Testing
H0: No significant difference in age across years
H1: Significant difference exists

Test: Independent T-Test
p-value: 0.059

Conclusion
Fail to reject H0. No significant difference found.

Tools Used
Python (Pandas, Matplotlib)
SQL
Power BI / Excel
